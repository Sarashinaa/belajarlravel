# Belajar Laravel

Repositori ini berisi kumpulan tugas mata kuliah Pemrograman Web menggunakan framework Laravel.

## Tentang Proyek

Proyek ini merupakan aplikasi Laravel yang digunakan untuk mengerjakan berbagai tugas praktikum mata kuliah Pemrograman Web. 

## Daftar Tugas

### Tugas Form (5 November)
- **File**: `form.blade.php` - Halaman form input
- **File**: `result.blade.php` - Halaman hasil form

### Tugas Route (22 November)
- **File**: `home.blade.php` - Halaman home/beranda
- **File**: `about.blade.php` - Halaman tentang saya

## Instalasi

1. Clone repositori ini
2. Install dependencies:
    ```bash
    composer install
    ```
3. Copy file `.env.example` ke `.env`
4. Generate application key:
    ```bash
    php artisan key:generate
    ```
5. Jalankan server:
    ```bash
    php artisan serve
    ```
